# Step 01 - Agent Skills and Discovery

## Welcome to Section 3: Enterprise Agentic Patterns

In **Section 3**, you'll build an intelligent **Customer Trip Planner** for Miles of Smiles — a system that helps customers plan road trips using enterprise-grade agentic patterns. 

### The Scenario

Miles of Smiles offers customers more than just a car — an intelligent Trip Planner helps plan the entire journey. Customers describe their trip ("7 days on the California coast, family of 4, budget $3000") and the system picks the right vehicle, plans a route, checks live conditions, and books accommodations.

---

## Agent Skills

Different trip types need different planning expertise. A "family beach vacation" skill knows about kid-friendly stops and rest frequency. A "business travel" skill optimizes for speed and airport proximity. The system loads the right skill based on what the customer describes.

In this step, you'll learn how to use `SkillsToolProvider` to dynamically load expertise from the filesystem, giving agents specialized knowledge without code changes.
